# 1314·一生一世 — 言和13周年 × 洛天依14周年 纪念网页

言和（7月11日·13周年）与洛天依（7月12日·14周年）双人生日纪念网页，13+14 = 一生一世。

## 本地预览

```bash
# 使用 Python 启动本地服务器
python -m http.server 8080 --directory .

# 或使用 Node (如已安装)
npx serve .
```

然后访问 `http://localhost:8080`

## 页面结构

| 模块 | 说明 |
|------|------|
| Hero 首屏 | 1314·一生一世，双人主题展示 |
| 角色档案 | 言和 & 洛天依 详细资料 |
| 时光轴 | 从出道至今的重要时刻 |
| 经典合唱 | 双人合唱曲目展示 |
| 1314 主题区 | 一生一世概念呈现 |
| 今年贺曲 | 粉丝同人贺曲 |
| 倒计时 | 实时倒计时到生日 |
| 祝福墙 | 粉丝留言互动 |
| 管理后台 | 按 Ctrl+Shift+A 进入 |

## 管理后台

- **入口**：在页面按 `Ctrl+Shift+A` 或访问 `/?admin`
- **密码**：`1314`
- **功能**：
  - 编辑角色档案（姓名、颜色、声源、简介等）
  - 增删改时间线事件
  - 增删改曲目（原创/翻唱/贺曲）
  - 删除祝福留言
  - 导入/导出全部数据
  - 重置为初始数据

## 技术架构

- 纯静态页面，零依赖
- ES Modules 模块化架构
- 数据驱动：JSON 文件 → data-engine → renderer → DOM
- 管理后台数据保存至 localStorage

## 数据文件

| 文件 | 内容 |
|------|------|
| `data/config.json` | 站点全局配置 |
| `data/characters.json` | 角色档案 |
| `data/timeline.json` | 时间线事件 |
| `data/songs.json` | 曲目列表 |
| `data/wishes.json` | 预置祝福 |

## 部署

推荐托管到 Vercel、Netlify 或 GitHub Pages，直接指向项目根目录即可。
